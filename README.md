# TPWineQuality

## Installation 
python3.8 -m venv .venv

source .venv/bin/activate

pip install --upgrade pip

pip install pandas

pip install -U scikit-learn

pip install "fastapi[all]"

## Launch
python3 main.py

## Test
pip install pytest

python3 -m pytest --cov
